# EXHOLD Website

Frontend-only Vite site for the ICT707 Assessment 4 EXHOLD business proposal.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy To Vercel

Import this `website` folder as a Vercel project.

- Framework preset: Vite
- Build command: `npm run build`
- Output directory: `dist`
